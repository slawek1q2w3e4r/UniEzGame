# UniEzGame

UniEzGame is a simple python package for creating 2D games using SDL2

## Installing SDL2:

### Widnows:
**wget https://github.com/libsdl-org/SDL/releases/download/release-2.30.8/SDL2-devel-2.30.8-VC.zip**

### Linux:
**wget https://github.com/libsdl-org/SDL/releases/download/release-2.30.8/SDL2-devel-2.30.8-VC.zip**

### MacOS
**wget https://github.com/libsdl-org/SDL/releases/download/release-2.30.8/SDL2-devel-2.30.8-VC.zip**
